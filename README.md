# analysis

a [Sails](http://sailsjs.org) application

# Install

```bash
sudo apt-get install nodejs
sudo npm install -g sails
sudo npm install -g coffee-script
npm install
```

# Test

```bash
grunt test
```

# Run

```bash
forever -w start app.js
```

# Logs

```bash
forever logs app.js -f
```

# Quit

```bash
forever stop app.js
```
